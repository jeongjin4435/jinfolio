# JIN:CODE - Developer Portfolio

AI·SW와 데이터사이언스를 배우며 개발자를 목표로 성장하는 이정진의 개인 포트폴리오 웹사이트입니다.

## 프로젝트 소개

이 프로젝트는 단순한 자기소개 페이지가 아니라, 여러 페이지로 구성된 개발자 포트폴리오 웹사이트입니다. 기존의 긴 스크롤형 구조가 아니라 컴퓨터 화면에서 보기 좋도록 Home, About, Experience, Projects, Roadmap, Contact 페이지를 분리했습니다.

포트폴리오의 핵심 방향은 콘텐츠 크리에이터가 아니라 개발자입니다. 다만 영상 편집, 스토리 구성, 시각적 표현, 경진대회, 스터디, 학생회와 동아리 활동 경험을 개발자로 성장하는 데 활용할 수 있는 강점으로 정리했습니다.

## 파일 구조

```text
jin-code-portfolio/
├── index.html
├── about.html
├── experience.html
├── activities.html
├── projects.html
├── roadmap.html
├── contact.html
├── style.css
├── script.js
└── README.md
```

`activities.html`은 기존 링크가 남아 있을 경우를 대비해 `experience.html`로 이동하도록 남겨두었습니다.

## 실행 방법

1. 프로젝트 폴더를 다운로드합니다.
2. `index.html` 파일을 브라우저에서 실행합니다.
3. 상단 메뉴를 통해 각 페이지로 이동합니다.

## GitHub Pages 배포 방법

1. GitHub에서 새 저장소를 생성합니다.
2. 위 파일들을 저장소에 업로드합니다.
3. 저장소의 `Settings` 메뉴로 이동합니다.
4. `Pages` 메뉴에서 Branch를 `main`으로 설정합니다.
5. 저장 후 생성된 배포 링크로 접속합니다.

## 느낀 점

이번 프로젝트를 통해 HTML, CSS, JavaScript의 역할을 나누어 이해할 수 있었습니다. 특히 여러 페이지로 구성된 포트폴리오를 만들면서 공통 CSS와 JavaScript를 어떻게 재사용할 수 있는지 배웠습니다.

또한 프로젝트 필터링, 모달창, 다크모드, localStorage 저장 기능을 구현하면서 JavaScript가 단순히 화면 효과를 주는 역할을 넘어 사용자의 행동에 따라 웹사이트를 동적으로 바꿀 수 있다는 점을 알게 되었습니다.

앞으로는 이 포트폴리오를 계속 개선하면서 실제 데이터 분석 프로젝트와 AI 활용 웹서비스를 추가하고 싶습니다.
