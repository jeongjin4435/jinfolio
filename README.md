# Jinfolio — AI로 이야기를 설계하는 크리에이티브 개발자

## 1. 프로젝트 소개

Jinfolio는 AI·SW 분야를 배우고 있는 이정진의 성장 과정, 관심 분야, 프로젝트, 미래 계획을 소개하는 인터랙티브 포트폴리오 웹사이트입니다.

단순한 자기소개 페이지가 아니라, JavaScript 기능을 활용해 프로젝트 필터링, 상세 모달창, 다크모드, 성장 목표 관리, AI × China 콘텐츠 아이디어 생성기 등을 구현했습니다.

## 2. 제작 목적

이 프로젝트의 목적은 HTML, CSS, JavaScript만 사용하여 GitHub Pages에 배포 가능한 완성도 높은 개인 포트폴리오 웹사이트를 제작하는 것입니다.

또한 저의 콘텐츠 기획 경험, 영상 편집 경험, 스토리텔링 능력, 중국어 콘텐츠 관심사를 AI·SW 학습 방향과 연결해 보여주고자 했습니다.

## 7. 실행 방법

1. 프로젝트 폴더를 다운로드합니다.
2. `index.html` 파일을 브라우저에서 엽니다.
3. 또는 VS Code의 Live Server 확장 프로그램을 사용해 실행합니다.

## 8. GitHub Pages 배포 방법

1. GitHub에서 새 저장소를 생성합니다.
2. `index.html`, `style.css`, `script.js`, `README.md` 파일을 업로드합니다.
3. 저장소의 `Settings` 메뉴로 이동합니다.
4. 왼쪽 메뉴에서 `Pages`를 선택합니다.
5. `Branch`를 `main`으로 설정하고 저장합니다.
6. 잠시 후 제공되는 GitHub Pages 링크로 접속하면 웹사이트를 확인할 수 있습니다.

## 9. 느낀 점

이번 프로젝트를 통해 단순히 화면을 꾸미는 것뿐만 아니라, JavaScript를 활용해 사용자가 직접 상호작용할 수 있는 기능을 구현하는 경험을 할 수 있었습니다.

특히 프로젝트 필터링, 목표 관리, LocalStorage 저장 기능을 만들면서 웹사이트가 단순한 소개 페이지가 아니라 실제로 사용할 수 있는 웹앱처럼 동작할 수 있다는 점을 배웠습니다.

앞으로는 AI API를 연결하여 실제로 콘텐츠 아이디어를 자동 생성하거나, 중국어 자막과 스토리보드를 만들어주는 기능으로 확장해보고 싶습니다.
